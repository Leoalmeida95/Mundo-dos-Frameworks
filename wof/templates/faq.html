Qual o propósito do Mundo dos Frameworks?

O objetivo do sistema é permitir que um usuário escolha, de uma lista de linguagens, um determinado framework sobre o qual o mesmo poderá aprender sobre, e passar suas experiências dele. O sistema pretende, portanto, fazer um levantamento de dados de variados frameworks para que, em um mesmo ambiente, o usuário possa ter diversas informações sobre qualquer Framework de seu interesse.

O que é uma "Hello World"?

Na programação, o "Hello World" ou "Olá Mundo" é o primeiro passo a ser dado por uma pessoa que está aprendendo uma nova Linguagem de Programação ou Framework. Ele se resume no mínimo necessário para que aquela determinada tecnologia funcione e "imprima" algo na tela. Assim, o Hello World no sistema representa o código básico que é preciso para que um determinado Framework funcione.

Quais são os critérios para decidir se algo é um "framework web"?

Digamos que se você puder terminar a frase "Acabei de criar esse novo aplicativo/sistema web com [insira o nome do framework aqui]", então ele provavelmente seria considerado uma framework web. Note que os frameworks não precisam ser "full-stack", eles podem ser apenas do lado do cliente ou apenas do lado do servidor. O Mundo dos Frameworks não inclui bibliotecas JavaScript como jQuery e MooTools, já que elas são normalmente usadas para aprimorar aplicativos da Web que são criados usando outro Framework.
